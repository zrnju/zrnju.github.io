WEB端 — Course+

项目介绍

Course+是一个面向研究生教育的在线辅导直播平台，用户可以在网站上选择心仪的老师，购买其考研辅导课程进行学习，学习方式有直播、独家资料下载等，旨在为志向考研的考生提供一个有信服力的平台。

目标人群

有志于报考南京大学研究生的各大考生，想要向已经录取的南大研究生学习但是受限于时间地点等问题不方便线下授课的考生。

我的贡献

这是我作为专职UI设计师做的第一个完整商业项目。我负责所有的视觉规范编写和视觉稿、宣传稿设计。同时，我和产品经理针对交互设计的合理性进行了很多讨论，也和前端开发进行技术上的实现方面的讨论，积累了大量实践经验。


项目背景


Course+项目是我在大三实习的公司里做的第一个完整的商业项目，属于B端产品。course+项目使用Scrum框架进行开发，Teambition进行管理，目标是两周内开发大概80%的功能。这对于产品经理、设计师和开发人员来说都是一个不小的挑战。

Srcum完整文档下载地址：http://zherry.top/Scrum.pdf



竞品分析


网易云课堂

优点：

  • 课程分类合理，面包屑导航方便用户导向；
  • 优秀导师介绍，打造名师效应；

缺点：

  • 首页内容太过繁杂，颜色太多容易疲劳；
  • 直播课程体验较差；
  • 播放器两种样式容易让人混乱；


腾讯课堂

优点：

  • 直播体验较好，情感化设计引起共鸣；
  • 首页简洁明快，专业感更好；

缺点：

  • 个人页面逻辑混乱，寻找课程较难；
  • 课程层级太多，操作层级太深；

Airbnb

优点：

  • 首页简洁，搜索设计符合用户心理；

缺点：

  • 收藏功能很难寻找；
  • 本地化完成程度低；



设计方案敲定

在和整个开发团队讨论完之后，我们决定使用蚂蚁金服设计平台的Ant作为前端开发框架。相应的，我选择使用ant.design 作为整个项目的设计语言。关于ANT DESIGN的更多细节可以参考官方网站。
